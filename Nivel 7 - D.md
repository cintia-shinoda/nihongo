# Nível D


### ```あります```
|    | FORMAL | INFORMAL |
|:---:|:---|:---|
| não pretérito afirmativo | 〜が　**あります** | 〜が　**ある** |
| não pretérito negativo | 〜が　**ありません** | 〜が　**ない** |
| pretérito afirmativo | 〜が　**ありました** | 〜が　**あった** |
| pretérito negativo | 〜が　**ありませんでした** | 〜が　**なかった** |
<br>
